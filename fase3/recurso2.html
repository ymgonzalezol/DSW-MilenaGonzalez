HTML con estilos CSS Bootstrap

 

 

En primer lugar, el CSS de Bootstrap incluye el reinicio que “normaliza” los estilos de página web. Los navegadores web tienen estilos diferentes, algo feos, que se aplican a las páginas web de forma predeterminada. Dado que los valores predeterminados de cada navegador son un poco diferentes, Bootstrap usa Reboot para anular los valores predeterminados del navegador y proporciona la misma “línea de base elegante y consistente” sobre la que se puede construir, independientemente del navegador. Esto significa que si la página web se ve en Safari, Chrome, Firefox o IE, comenzará con los mismos estilos de línea base de CSS.
